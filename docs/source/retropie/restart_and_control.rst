System and Emulator Control
===========================

Restart Current Game
--------------------

Keyboard:
- Press F1 → select "Restart"

Controller:
- Use RetroArch hotkey combination (configured in settings)

Restart RetroArch
-----------------

1. Open RetroArch menu (Hotkey + X or equivalent).
2. Navigate to:
   Main Menu → Quit → Restart RetroArch

Restart EmulationStation
------------------------

CLI:
  emulationstation --restart

Or via menu (if not in kiosk mode).

Reboot System
-------------

CLI:
  sudo reboot

Shutdown System
---------------

CLI:
  sudo shutdown now
