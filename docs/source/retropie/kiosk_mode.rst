Kiosk Mode
==========

Exiting Kiosk Mode
------------------

1. Use the configured button combination (see Google Doc).
2. Enter passcode if prompted.

Entering Kiosk Mode
-------------------

1. Open EmulationStation menu.
2. Navigate to:
   UI Settings → Kiosk Mode
3. Enable Kiosk Mode.
4. Confirm lock.

Notes
-----
- Kiosk mode restricts access to settings and file manipulation.
- Only exit when necessary.
