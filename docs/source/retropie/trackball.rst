Trackball Configuration
=======================

Overview
--------

Some games require trackball input instead of standard joystick (retropad).

Reference Games
---------------
- Golden Tee (working)
- Centipede (working)

Configuring Trackball for New Games
-----------------------------------

1. Launch a working reference game.
2. Open RetroArch menu.
3. Navigate to:
   Quick Menu → Controls
4. Note:
   - Device Type
   - Axis mappings

Apply to New Game
-----------------

1. Launch target game.
2. Open RetroArch menu.
3. Set:
   Device Type → Mouse or Trackball equivalent
4. Adjust sensitivity under:
   Settings → Input → Analog Sensitivity

Save Configuration
------------------

- Per game:
  Quick Menu → Overrides → Save Game Overrides

Notes
-----
- Trackball usually maps as a mouse device.
- Incorrect mapping results in no movement or erratic behavior.
