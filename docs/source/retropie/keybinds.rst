RetroArch Keybinding
====================

Global Keybinding
------------------

1. Open RetroArch menu (Hotkey + X).
2. Navigate to:
   Settings → Input → Port 1 Binds
3. Change desired bindings.
4. Save:
   Configuration File → Save Current Configuration

Per-Core Keybinding
-------------------

1. Load a game using the target core.
2. Open RetroArch menu.
3. Change bindings.
4. Save:
   Quick Menu → Overrides → Save Core Overrides

Per-Game Keybinding
-------------------

1. Load the specific game.
2. Open RetroArch menu.
3. Modify controls.
4. Save:
   Quick Menu → Overrides → Save Game Overrides

RetroPie-Level Input Configuration
----------------------------------

1. Exit to terminal or EmulationStation menu.
2. Run:
   sudo ~/RetroPie-Setup/retropie_setup.sh
3. Select:
   Configuration / Tools → Configure Input
4. Follow prompts.

Notes
-----
- RetroPie config affects EmulationStation and initial mappings.
- RetroArch overrides take precedence during gameplay.
